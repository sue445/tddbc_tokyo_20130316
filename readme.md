## 1枚目
```
class LTSV {
  set(String key, String val) : String
  get(String key) : String
  dump() : String
}

ltsv = new LTSV();
ltsv.set( "foo", "hoge" ) #=> null
ltsv.get( "foo" )         #=> "hoge"
ltsv.dump()               #=> "foo:hoge\n"

ltsv.set( "bar", "fuga")  #=> null
ltsv.dump()               #=> "foo:hoge\tbar:fuga\n"

ltsv.set( "foo", "piyo")  #=> "hoge"
                          #     ↑ 置き換えられた値

ltsv.dump()               #=> "bar:fuga\tfoo:piyo\n"
                          #     ↑ 格納順にダンプされる
```
